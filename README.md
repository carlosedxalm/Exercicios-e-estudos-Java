# Exercícios e Estudos em Java

Este repositório reúne exercícios práticos e estudos desenvolvidos durante o aprendizado da linguagem **Java**. O objetivo principal é consolidar conceitos fundamentais de lógica de programação e programação orientada a objetos, registrando a evolução por meio de aulas e exercícios de fixação.

Cada pasta representa uma **aula concluída** ou um **conjunto de exercícios**, sempre focando em um tema específico.

---

## 📁 Estrutura do Repositório

### 📂 EntradaDeDados

Contém exemplos e exercícios sobre **entrada e saída de dados** em Java, utilizando principalmente a classe `Scanner`. São abordados:

* Leitura de números inteiros e decimais
* Leitura de strings
* Exibição de dados no console

Esses exercícios ajudam a entender a interação básica entre o usuário e o programa.

---

### 📂 EstruturaCondicional

Exercícios voltados ao uso de **estruturas condicionais**, como:

* `if`
* `else`
* `else if`

Os códigos resolvem problemas simples de tomada de decisão, reforçando a lógica condicional.

---

### 📂 EstruturaDeRepeticao

Nesta pasta estão exemplos e exercícios relacionados a **laços de repetição**, incluindo:

* `for`
* `while`
* `do-while`

Os exercícios praticam repetições controladas por contadores e condições, comuns em problemas do dia a dia.

---

### 📂 ExercicioDeFixacao1

Conjunto de exercícios propostos para **fixação dos fundamentos iniciais**, abordando:

* Variáveis
* Operadores aritméticos
* Estruturas condicionais
* Estruturas de repetição

Esses exercícios ajudam a consolidar a base da lógica de programação em Java.

---

### 📂 ExercicioEstruturaCondicional

Exercícios específicos para reforçar o uso de **condicionais**, focando na resolução de problemas práticos que exigem decisões lógicas mais elaboradas.

---

### 📂 ExercicioDeFixacaoOrientObjetos

Pasta dedicada a introdução dos conceitos de **Programação Orientada a Objetos (POO)**, incluindo:

* Criação de classes
* Atributos
* Métodos
* Instanciação de objetos
* Encapsulamento

Esses exercícios marcam a transição da lógica procedural para o paradigma orientado a objetos.

---

## ▶️ Como Executar os Exercícios

1. Clone o repositório:

```bash
git clone https://github.com/carlosedxalm/Exercicios-e-estudos-Java.git
```

2. Abra o projeto em sua IDE Java preferida:

* IntelliJ IDEA
* Eclipse
* VS Code

3. Navegue até a pasta desejada
4. Execute a classe que contém o método `main`
